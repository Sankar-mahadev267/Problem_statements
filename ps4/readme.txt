4. Problem Statement: Minutes to days. Hours and minutes Conversion

You are. tasked with Creating a program that talker a Positive integer representing a total number of minutes as input and converts it into days, hours and remaining minutes. The goal is to provide a user friendly representation of a given duration in terms of the number of full days full hours, and any remaining minutes.

Input: The Program Should accept a positive integer, n, where 1<=n<=10^7

output: The Program should display the equivalent duration in days, hours & minutes.

Conversion rules: 
• Assume that each day has 24 hours and each hour has 6o minutes.
• when Converting minutes to hours, consider an hour as having 60 minutes.
• A day is considered to have 24 hours.
